Exclude specific payment methods for selected WooCommerce products and product categories.

== Description ==

WooCommerce Product Exclude Payment is a plugin that allows you to exclude specific payment methods for selected WooCommerce products and product categories. This is useful when you have certain products or product categories that require specific payment options.

Features:

* Exclude payment methods for specific products
* Exclude payment methods for specific product categories
* Easy to use, no coding required

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/woocommerce-product-exclude-payment` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Go to the product or product category edit screen and select the payment methods you want to exclude.

== Frequently Asked Questions ==

= Does this plugin require WooCommerce to be installed and activated? =

Yes, this plugin requires WooCommerce to be installed and activated.

= Can I exclude payment methods for multiple products or product categories at once? =

Yes, you can exclude payment methods for each product or product category individually.
